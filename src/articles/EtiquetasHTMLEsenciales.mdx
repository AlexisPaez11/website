## ¿Qué es HTML?
HTML es un lenguaje de marcado que define la estructura del contenido de un sitio web. Consiste en una serie de elementos que son utilizados para encerrar diferentes partes del contenido con el objetivo de que se vean o se comporten de una cierta manera. Generalmente se utilizan otras tecnologías como (CSS) para la apariencia de un sitio web o (JavaScript) para funcionalidad/comportamiento.

<div class='espacio'/>

## ¿Qué son las etiquetas HTML?
Cuando escribes código en HTML, estás escribiendo etiquetas HTML.

Todas las etiquetas HTML están hechas con un número de partes específicas, incluyendo:

- El carácter 'menor que': &lt;
- Una palabra o carácter que determina qué etiqueta se está escribiendo
- Cualquier número de atributos HTML que se quiera usar: nombre='valor'
- El carácter 'mayor que': &gt;
<div class='lenguaje'><SiHtml5 /></div>
```html
<h1 class='title'>Lorem ipsum</h1>
```

## ¿Cuáles son las etiquetas HTML básicas?

### &lt;head&gt; &lt;head&gt;
La etiqueta &lt;head&gt; es fundamental en la estructura del documento HTML. Es una sección informativa que el navegador no le muestra al usuario.

Los elementos que podemos encontrar en esta sección incluyen el &lt;title&gt;, las etiquetas meta &lt;meta&gt; que describen el documento y referencias a scripts, estilos como &lt;style&gt;, &lt;script&gt;, &lt;link&gt;.

### &lt;body&gt; 
La etiqueta <body/> hace referencia al cuerpo del contenido de un documento HTML. Es una etiqueta esencial, ya que indica donde empieza el contenido visible para el usuario.

### &lt;div&gt;
La etiqueta <div/> nos permite definir un bloque o división de contenido y se utiliza para componer elementos (párrafos, encabezados, listas, tablas, divisiones, etc).

### &lt;h1&gt; ... &lt;h6&gt;
Los encabezados HTML son etiquetas que se utilizan para definir títulos y subtítulos para estructurar el contenido, la etiqueta <h1/> es el más relevante y la etiqueta <h6/> la menos relevante, de esta manera ayudamos al navegador a entender la importancia de nuestro contenido.

### &lt;p&gt;
La etiqueta <p/> nos sirve para agrupar texto dentro de un párrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.

### &lt;a&gt;
La etiqueta &lt;a&gt; nos permite realizar enlaces hacia documentos externos, como otros sitios web, direcciones de correo o cualquier otra URL a través del atributo href donde indicaremos el enlace al que queremos conectar y con el atributo target podemos indicar que el enlace se abra en una nueva ventana. También nos permite enlazarnos hacia alguna parte específica del mismo documento.

```html
<a href='https://www.google.com/' target='_blank'>Enlace a Google</a>
```

### &lt;ol&gt; &lt;ul&gt; &lt;li&gt;

La etiqueta &lt;strong&gt; se utiliza para definir dentro del documento, texto que es considerado importante. Esto produce en el navegador el efecto de aplicar negrita para dar énfasis.
